# 🖥️ Mini ERP Desktop

Sistema ERP Desktop desenvolvido como Trabalho de Conclusão de Curso (TCC), com foco em gestão comercial simplificada para pequenas empresas.

---

## 📌 Sobre o Projeto

O **Mini ERP Desktop** é um sistema desenvolvido para auxiliar no gerenciamento de processos comerciais, oferecendo controle e organização das seguintes operações:

- 📦 Cadastro de Produtos  
- 👤 Cadastro de Clientes  
- 💰 Controle de Vendas  
- 📉 Controle de Baixas (Fiado)  
- 📊 Registro de Movimentações  

O projeto tem como objetivo aplicar boas práticas de desenvolvimento, organização de código, modelagem de banco de dados e padronização de commits.

---

- C#

## 🗄️ Padrões Adotados

### 📌 Banco de Dados

- Nomes de tabelas no singular  
- Nomes descritivos e sem abreviações  
- Chave primária padronizada como `Id`  
- Chaves estrangeiras no formato `NomeTabelaId`  

---

## 📝 Padronização de Commits

O projeto segue um padrão organizado de commits utilizando emojis para facilitar a identificação das alterações:

| Tipo | Descrição | Exemplo |
|------|------------|----------|
| 💄 feat | Estilização / nova funcionalidade visual | 💄 feat: Ajuste no layout da tela de vendas |
| 📚 docs | Documentação | 📚 docs: Atualização do README |
| 🚚 chore | Mover / Renomear arquivos | 🚚 chore: Renomeando pasta Services |
| 🗑️ remove | Remoção de arquivos | 🗑️ remove: Removendo arquivo antigo |

### ✅ Exemplo Oficial de Commit

```
📚 docs: Atualização do README
```

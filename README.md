# Mini ERP Desktop - TCC

Sistema Desktop para Gestão de Estoque e Vendas  
Desenvolvido como Trabalho de Conclusão de Curso (TCC)

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de um sistema desktop para controle de:

- Cadastro de Clientes
- Cadastro de Produtos
- Controle de Estoque
- Registro de Vendas
- Controle de Contas a Receber (Fiado)

O sistema foi desenvolvido utilizando arquitetura em camadas e metodologia Scrum.

---

## 🛠 Tecnologias Utilizadas

- C#
- .NET 8
- WinForms
- SQLite
- Arquitetura em Camadas
- Metodologia Scrum

---

## 🏗 Estrutura do Projeto

O projeto está organizado em camadas:

- UI (Interface do Usuário)
- Application (Regras de Negócio)
- Domain (Entidades)
- Infrastructure (Acesso a Dados)

---

## 📊 Funcionalidades

✔ Cadastro de Clientes  
✔ Cadastro de Produtos  
✔ Controle de Estoque  
✔ Registro de Vendas  
✔ Controle de Contas a Receber  
✔ Tela de Baixa de Pagamento
